Time, Space, Complexity

Time complexity is a measure of the amount of time an algorithm takes to run as a function of the size of its input. An efficient algorithm will have a lower time complexity, ensuring faster execution even with large inputs.

Space Complexityrefers to the amount of memory space an algorithm needs relative to the input size. For example, if an algorithm uses O(n) extra space, it means the memory consumption grows linearly with the input size.
